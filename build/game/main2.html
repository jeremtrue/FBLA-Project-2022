Wordle Project

Please read instructions before playing.



Instructions:
To play: the user guesses a five letter word, not random letters.
The words will change a color.
Gray if the letter is not in the word.
Yellow if the word has the letter but it is in the wrong place.
Green if the letter is right and in the right spot.
Keyboard is supported.
