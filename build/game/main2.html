Wordle Project

Read instructions before playing.



Instructions: To play: the user guesses a five letter word.
The words will change a color.
Gray if the letter is not in the word.
Yellow if the word has the letter but it is in the wrong place.
Green if the letter is right and in the right spot.
When you type the keyboard letters also change that correspond with the letters in the Boxes.
