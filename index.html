Wordle Unleashed

Wordle Project

Please read instructions before playing.



Instructions:
Guess a five letter word, not random letters.
Press Enter and watch the colors of the letters.
Gray if not in the words.
Yellow if the word has the letter but it is in the wrong place.
Green if the letter is right and in the right spot.
Keyboard and On-Screen input works!
